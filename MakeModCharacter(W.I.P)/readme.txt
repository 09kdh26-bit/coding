사용 방법
메모장에서 [파일] -> [다른 이름으로 저장]을 눌러.

파일 형식을 '텍스트 문서'에서 '모든 파일'로 변경해.

파일 이름을 MakeMod.bat (끝에 반드시 .bat이 들어가야 해) 로 적고 저장해.

원본 바닐라 파일들이 들어있는 vanilla_musketeer 폴더를 MakeMod.bat 파일과 같은 곳에 둬.

MakeMod.bat 파일을 더블 클릭해.

검은색 창이 뜨면서 순식간에 파일 복사, 내부 코드 치환(`` 같은 파일 내용 포함), 파일명 변경까지 완료하고 mod_stargrit이라는 새 폴더를 짠! 하고 만들어 줄 거야. 코드를 보면 대소문자(musketeer, Musketeer, MUSKETEER)도 구분해서 치환하도록 짜두었으니 기존 자바 프로그램보다 훨씬 더 깔끔하고 버그 없이 작동할 거야.
